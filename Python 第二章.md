# 2025/10/9 第二章 变量和简单数据类型

## 变量

在 Python 中，**变量**是一个指向值的引用，无需在使用前声明数据类型。在对变量进行初始化和操作时，解释器会自动推断其类型。

**变量名**只能包含**字母**、**数字**和**下划线**，不能包含空格或其他特殊字符。

### 理解

在 Python 中使用变量时，无需担心数据类型不匹配，解释器会自动推断最适合的类型。程序员有义务保持变量名的一致性，避免因拼写错误导致程序出错。

## 方法

**方法**是一种与对象关联的函数，它依赖于对象（通常是类的实例）来调用。与普通函数不同，方法必须在对象上调用。

例如：

```python
message = "AUG"
print(message.title())
```

输出结果为：

```
Aug
```

这里 `title()` 是一个字符串方法，它将每个单词的首字母转换为大写。

## 在字符串中使用变量

在 `print(f"{变量名}")` 中，`f"..."` 被称为 **f-string**（格式化字符串），其中 `f` 是 `format` 的缩写。Python 会将花括号 `{}` 中的变量替换为其实际值。

## 转义字符

Python 中的转义字符（如 `\n`、`\t` 等）与 C 语言类似。

## Python 的某些特质

1. **数字中的下划线**：  
   在 Python 中，`1_000_000_000` 与 `1000000000` 完全相同，下划线仅用于提高可读性。

2. **多重赋值**：  
   可以使用 `x, y, z = 1, 2, 3` 的方式一次性为多个变量赋值。Python 会按顺序将值赋给对应的变量，这种语法在 C 语言中是不被允许的。
